# FORÇA MILITAR BRASILEIRA - FMB

Sistema militar desenvolvido para gerenciamento de membros, promoções, treinamentos, controle de atividades e administração da Força Militar Brasileira.

---

## 🇧🇷 Sobre o Sistema

A Força Militar Brasileira (FMB) é uma organização baseada em disciplina, hierarquia e comprometimento.

O sistema foi criado para fornecer uma plataforma moderna e profissional para gerenciamento completo da corporação.

### Recursos

- Sistema de Login
- Sistema de Cadastro
- Hierarquia Militar
- Promoções
- Rebaixamentos
- Controle de Treinamentos
- Controle de Atividades
- Hall da Fama
- Ranking de Instrutores
- Ranking de Atividade
- Sistema de Medalhas
- Logs Administrativos
- Dashboard Operacional
- Integração com Habbo
